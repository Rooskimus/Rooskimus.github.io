Working Together Live Project

Live Project Retrospective

As part of the curriculum at the Tech Academy I had the opportunity to work on a real project as a team of new developers. During the Live Project we worked on a future update to the Tech Academy's website, which is an Entity-Framework Code-First MVC site. Using the Azure DevOps site we took on numerous user stories, collaborating with each other to create and add robust Job Placement functionality to the site for future students.


During this project I learned a lot about the workflow of an AGILE project and working as a team. It also helped deepen my understanding of many concepts, especially MVC Code-First, JSON, working with APIs, and much more. I did a detailed code retrospective, so be sure to give it a look on my GitHub.

Back-End C# Projects

Twenty One

In this project I created a working, text-based Black Jack game following a video tutorial series. It was designed to cover many concepts, including creating and using libraries, inheritance, logging to .txt files and databases, exception handling, and many more! There were a few places where I chose to go a different route from the tutorial, mainly in how hand values are determined as well as in the implementation of logging. It was a lot of fun to make and I really enjoyed putting my own twist on it.


Additionally I worked on many other proof-of-concept drills. I usually added something above and beyond the requirements to satisfy my own curiosity and make them more interesting!


SQL Server

Another integral part of being a software developer is handling database transactions. The language I practiced in was Microsoft's SQL Server. Using SQL Server's Management Studio I practiced creating and managing databases, creating stored procedures, adding and editing tables, assigning primary and foreign keys and more.
